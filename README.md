## About
基于 Vue3 (vite) 实现艾柏霖科技官网, 提供关于我们/新闻列表/应用/产品列表/联络我们表单等

## Feature

* 使用 axios http请求库 与 element-plus 组件库
* 使用 lodash-es js工具库
* 使用 i18n 实现多语言切换
* 支持多环境开发配置 .env/.env.test/.env.prod

## Usage
```
# 执行
pnpm run dev
```
